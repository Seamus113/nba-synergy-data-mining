# nba-synergy-data-mining
Data mining project analyzing player synergy in NBA lineups using association rule mining and lift.
## Dataset 
This project uses a publicly available NBA play-by-play dataset from Kaggle: https://www.kaggle.com/datasets/xocelyk/nba-pbp The dataset is based on Sports Reference play-by-play data and includes imputed on-court lineups for each play. Each record contains the five players on the court for both teams, along with game events and score updates. Because substitution data between periods is not fully available in public sources, the lineup information is reconstructed using in-game events and substitutions. The dataset covers NBA games from 1999 to 2022. 
## Notes 
Due to file size limitations, the full dataset is not included in this repository. Please download it directly from Kaggle using the link above.
